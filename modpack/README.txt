DISPATCH MOD PACK
=================

Three folders, one per setup. Download the mods for a setup and drop every
archive straight into its folder — unsorted, original names, still zipped.
Dispatch works out which archive is which mod by its file name.

  1 - Normal     LSPDFR and its essentials
  2 - Plugins    the full plugin lineup
  3 - Realism    the realism layer (defined later)

A mod shared by more than one setup only needs to be dropped in ONCE, in any
folder — the match looks across all three.

At install time only the mods the user ticks are unpacked; the rest of the
pack is never touched, so no unused files reach the game folder.

Fetched automatically from GitHub — do NOT put these in the pack (they are
version-locked and pulled fresh to match the game build):

  - Script Hook V .NET  (scripthookvdotnet/scripthookvdotnet)
  - RAGENativeUI  (alexguirre/RAGENativeUI)
  - LemonUI  (justalemon/LemonUI)
